Compress a raw image into a bitstream.

Parameters
[in]opCodeOpcode associated with this command invocation.
[in]cmdSeqCommand sequence number for this invocation.
[in]input_filePath to the input .raw image file.
[in]output_dirDirectory where the output .bin file is written.
[in]aelAbsolute error limit used for compression.
[in]override_xOverride image X dimension, or implementation-defined default when unused.
[in]override_yOverride image Y dimension, or implementation-defined default when unused.
[in]override_zOverride image Z dimension, or implementation-defined default when unused.
[in]override_dtypeOverride sample data type string (for example, u16).
[in]image_sample_lenNumber of image samples available in the component input buffer.
Precondition
input_file and output_dir are valid command arguments.
image_sample_len is less than or equal to kImageSampleBufElems.
Postcondition
A command response is emitted indicating success or failure.
